주요기능
1. 마크다운 에디터 (왼쪽) + 실시간 미리보기 (오른쪽)
2. 새 문서 / 저장 / 다른 이름으로 저장 / 불러오기 (.md)
3. 문서 메타데이터 (제목,작성일, 태그)
4. HTML로 내보내기 (Export as HTML)
5. 파일 목록 (프로젝트 폴더)을 왼쪽 사이드바에 표시

확장(추후)
- 이미지 드래그 & 드롭 (로컬 복사 및 상대 경로 삽입)
- 마크다운 템플릿 (블로그 포스트 헤더 자동 생성)
- 자동 저장 / 버전 관리 (간단한 undo/repo 외에 스냅샷)
- 프리셋 테마 (라이트/다크), 코드 하이라이팅
- publish 기능 : 블로그로 포스트 업로드(연동)

UI / 화면 구성
- 왼쪽 사이드바 : 파일트리 / 새 문서 버튼 / 설정
- 중앙 : 에디터 (textArea or CodeMirror)
- 오른쪽 : 렌더된 미리보기 (react-markdown)
- 상단 툴바 : 저장, 내보내기, 메타데이터(제목/태그) 입력

개발 마일스톤 (순서)
1. 프로젝트 초기화 : Electron + React 템플릿 구성
2. 에디터와 미리보기 동기화(실시간 렌더)
3. 파일 저장/불러오기 (로컬 .md)
4. 문서 메타데이터 UI / 파일 목록 사이드바
5. 내보내기 (HTML) 및 기본 스타일 적용
6. (옵션) 이미지 업로드 ,템플릿, 배포
